Git

  • $ git init {имя проекта} - создать новый локальный репозиторий
  • $ git clone {url-адрес} - скачать удаленный репозиторий с заданным именем
  • $ git status - просмотр изменений и создание коммитов (фиксация изменений)
  • $ git add {имя файла} - добавить файл в отслеживаемые файлы
  • $ git commit -m "сообщение" - сделать коммит (операцию записи изменений в репозиторий Git)
  • $ git push - отправить локальные коммиты в удаленный репозиторий
  • $ git log - вывести список коммитов
  • $ git pull - скачать изменения из удаленного репозитория
